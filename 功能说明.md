## 🚀 快速开始

### 启动系统

1. **确保 MT5 终端已启动并登录账户**
2. **在项目目录运行**：
   ```bash
   py main.py
   ```
3. **打开浏览器访问**：http://localhost:8000

### 界面功能说明

#### 仪表板
- 左上角：账户信息卡片（余额、净值、盈亏、持仓数）
- 左侧：快速交易面板（品种选择、价格、快速多空）
- 右侧：当前持仓列表
- 下方：挂单列表、系统信息

#### 手动交易
- 选择订单类型：市价单/限价单
- 输入交易参数：手数、止损点数、止盈点数
- 选择品种
- 点击做多或做空

#### 信号测试
- 输入策略参数
- 模拟发送 TradingView Webhook
- 查看响应结果

## 📊 Pine Script 示例

```pine
//@version=5
strategy("黄金交易策略", overlay=true, default_qty_type=strategy.cash, default_qty_value=1000, commission_type=strategy.commission.percent, commission_value=0.1)

// 策略逻辑（示例：均线交叉）
fastMA = ta.sma(close, 5)
slowMA = ta.sma(close, 20)

// 交易条件
longCondition = ta.crossover(fastMA, slowMA)
shortCondition = ta.crossunder(fastMA, slowMA)

// 执行交易
if (longCondition)
    strategy.entry("Long", strategy.long)
if (shortCondition)
    strategy.entry("Short", strategy.short)

// 止盈止损
strategy.exit("Exit", "Long", limit=close + 50, stop=close - 30)
strategy.exit("Exit", "Short", limit=close - 50, stop=close + 30)

// Webhook 警报（复制到 TradingView 警报设置中）
alertcondition(longCondition, title="做多信号", message='{"strategy":"MA_Cross","action":"buy","symbol":"XAUUSD","lot":0.1,"sl_points":30,"tp_points":50}')
alertcondition(shortCondition, title="做空信号", message='{"strategy":"MA_Cross","action":"sell","symbol":"XAUUSD","lot":0.1,"sl_points":30,"tp_points":50}')
```

## ⚠️ 注意事项

1. **MT5 必须先启动**：Python 程序无法替代 MT5 终端，需要先手动打开并登录
2. **先在模拟账户测试**：真实交易前务必在模拟账户充分测试
3. **合理设置止损**：交易有风险，务必设置止损
4. **控制仓位大小**：避免重仓，合理设置手数
5. **监控系统运行**：定期查看日志和通知
